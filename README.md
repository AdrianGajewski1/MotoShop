# MotoShop
A automotive shop made using Asp.Net Core and Angular 9.

# Technologies Used:
 * JWT
 * AutoMapper
 * Angular Material
 * XUnit
 * Moq
 * Google Authentication
 
 # Requirements
   * Dotnet SDK
   * Npm CLI
 
 # Usage
  * WebAPI
      * On the project root directory run
      ```
      dotnet build
      ```
       then
      ```
      dotnet run
      ```
  * Angular frontend
     * On the project src directory run
      ```
      npm start
      ```
  Then go to http://localhost:4200
